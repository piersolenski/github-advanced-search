# GitHub Advanced Search

Searches GitHub with some advanced options.

## Development

1. Clone repository

```
git clone git@github.com:piersolenski/favourite-playlists.git
```

2. Install dependencies

```
npm install
```

3. Run development

```
npm run dev
```

## Production

```
npm run build
```
